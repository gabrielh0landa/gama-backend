#  Gama Marketplace - Backend

Bem-vindo à API do projeto **Gama Marketplace**. Este ecossistema é composto por esta API, um [Frontend Web em Angular](https://github.com/gabrielh0landa/gama-frontend) e uma interface [Mobile em Kotlin/Android](https://github.com/gabrielh0landa/gama-mobile).

Atualmente, estamos em fase de desenvolvimento do **MVP**. O foco inicial é a estruturação do banco de dados e a gestão de lojas e seus respectivos produtos.
